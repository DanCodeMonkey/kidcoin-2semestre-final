# KidCoin - Documentação
O KidCoin é um projeto desenvolvido pelos alunos do curso de Desenvolvimento de Software Multiplataforma da FATEC - Votorantim.
A documentação completa pode ser acessada na pasta @docs do projeto.

## RESUMO

O Kidcoin é uma solução digital, estruturada no modelo Software as a Service (SaaS), desenvolvida para abordar a lacuna na educação financeira de crianças (8 a 10 anos) no Brasil. O projeto propõe uma plataforma web gamificada, inspirada em micro-aprendizagem, que transforma conceitos financeiros essenciais – como poupar e planejar – em uma experiência lúdica e interativa, atuando de forma preventiva contra o endividamento familiar. A funcionalidade do sistema é sustentada pela Gamificação e pela Trilha de Conhecimento para o engajamento dos alunos, e pela Gestão Escolar, que oferece aos professores ferramentas para acompanhamento de desempenho e cadastro de turmas, alinhando-se aos Objetivos de Desenvolvimento Sustentável (ODS) da ONU ao promover educação de qualidade e responsabilidade social.
